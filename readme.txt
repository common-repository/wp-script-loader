== Description ==

Wp Script Loader is used to upload script files i.e. js, css files.
If you are not a developer then it will be very difficult to upload css or js files to your wordpress site. You can use this plugin to upload these files.
You can upload jquery plugins or other files i.e. jqury-ui, bootstrap.

== Installation ==

Way 1:
1. Upload `wp-script-loader.zip` to the `/wp-content/plugins/` directory
2. Extract the zip file.
3. Activate the plugin from wordpress plugins page. And you have done.

Way 2: 
1. Go to plugins page in admin.
2. Click on Add New button.
3. Search for "Wp Script Loader".
4. Click install on Wp Script Loader.
5. Activate the plugin.
